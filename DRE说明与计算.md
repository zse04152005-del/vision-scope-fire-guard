# 我们项目的 DRE 计算过程（基于当前数据）

## 1. 数据口径
- `E_i`：测试阶段发现的问题数（以 `results/events.csv` 中“有效告警记录”条数作为近似）
- `E_(i+1)`：试运行阶段新增问题数（当前未记录，暂取 0）
- `D`：交付后缺陷数（当前未记录，暂取 0）

## 2. 当前统计值
- `E_i = 3`
- `E_(i+1) = 0`
- `D = 0`

## 3. 计算
**活动级 DRE：**
```
DRE_i = E_i / (E_i + E_(i+1))
      = 3 / (3 + 0)
      = 1.0
```

**总体 DRE：**
```
DRE = E / (E + D)
    = 3 / (3 + 0)
    = 1.0
```

## 4. 需要你补充后可得“正式 DRE”的数据
- 试运行阶段新增问题数 `E_(i+1)`
- 交付后缺陷数 `D`

只需替换上面的两个值即可得到最终结果。

# Usability（Learning Time）说明与计算（项目版）

## 1. 指标定义
- **Learning Time**：新用户在“无外部帮助”的情况下，从首次打开系统到能独立完成核心任务所需的时间。

## 2. 本项目核心任务（建议）
- 启动系统并进入监控界面  
- 添加/选择摄像头（USB 或 RTSP）  
- 启动检测并能看到实时画面  
- 理解并处理一次告警（查看告警详情）

## 3. 计算方式
- 记录每位用户完成以上任务的时间：`T1, T2, ..., Tn`
- 学习时间（平均）：  
```
Learning Time = (T1 + T2 + ... + Tn) / n
```
